This is a Venia website code library. This website built with ReactJs library, Redux, HTML, SASS, and ES6.

1. ReactJs - V18.1.0
2. Redux - V8.0.2
3. HTML5/SASS
4. Javascript - ES6

1. Nodejs - V16.14.2
2. NPM - V8.6.0
